מה זה עושה

What it does

🛡️

זיהוי מיידי

Instant detection

מזהה כל תשובת HTTP 418 — האות של נט פרי לחסימה — בדיוק כשהדפדפן מקבל אותה.

Detects every HTTP 418 — NetFree's block signal — the moment the browser receives it.

🔢

Badge חכם

Smart badge

מונה חסימות משמעותיות (לא פרסומות/מעקב) — ירוק כשנקי, אדום כשיש בעיה אמיתית.

Counts meaningful blocks only (ads & trackers excluded) — green when clean, red when it actually matters.

🎫

פתח בקשה בלחיצה

One-click requests

פותח טופס וייטליסט ממולא מראש ישירות באתר נט פרי.

Opens a pre-filled NetFree whitelist ticket right from the popup.

🌐

עברית ואנגלית

Hebrew & English

ממשק דו-לשוני מלא עם תמיכה ב-RTL.

Fully bilingual interface with RTL support.

🔒

פרטיות קודם

Privacy first

הכל נעשה מקומית בדפדפן שלך. שום URL לא נשלח לשרתים שלנו.

Everything runs locally in your browser. Not a single URL leaves your machine.

⚡

רשימה מתעדכנת

Live-updated list

רשימת הפרסומות/מעקב הנפוצים מתעדכנת אוטומטית ללא צורך בהתקנה מחדש.

The ad & tracker exclusion list auto-refreshes daily — no re-install needed.

סוגי חסימות שהתוסף מזהה

Block types detected

🚫 רשימה שחורה 🚫 Blacklisted — האתר נסקר ונחסם במפורש — site reviewed and explicitly blocked
⏳ לא ברשימה הלבנה ⏳ Not whitelisted — ממתין לאישור / בסקירה — pending review
⚙️ הגדרות אישיות ⚙️ Personal settings — המשתמש חסם את זה בהגדרות שלו — user's own filter settings
📁 סוג קובץ 📁 File type — סוג קובץ שלא נתמך בסינון אוטומטי (zip/exe וכו׳) — file type not supported by automatic filtering (zip/exe/etc.)
❓ צד שלישי ❓ Third-party — תת-משאב שנחסם, סוג לא מזוהה — sub-resource block, type undetectable
