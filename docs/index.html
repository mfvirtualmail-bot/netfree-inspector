מה זה עושה

What it does

🛡️

זיהוי מיידי

Instant detection

מזהה כל תשובת HTTP 418 — האות של נט פרי לחסימה — בדיוק כשהדפדפן מקבל אותה, בכל טאב.

Detects every HTTP 418 — NetFree's block signal — the moment the browser receives it, on every tab.

✨

בקשת וייטליסט במילוי אוטומטי

Auto-filled whitelist tickets

לחץ "פתח בקשה" — התוסף בונה נושא וגוף הבקשה הכוללים את כל החסימות בדף, פותח את הטופס בנט פרי, וממלא את השדות אוטומטית. לא צריך להעתיק ולהדביק.

Click "Open Request" — the extension assembles a full ticket subject + body listing every block on the page, opens the form on NetFree, and fills the fields automatically. No copy-paste needed.

🔢

Badge חכם

Smart badge

מונה חסימות משמעותיות (לא פרסומות/מעקב) — אייקון ירוק כשנקי, אדום כשיש בעיה אמיתית, עם מספר מדויק.

Counts meaningful blocks only (ads & trackers excluded) — green icon when clean, red with an exact count when it actually matters.

🎯

סינון לפי נתיב URL

Path-aware filtering

לא רק לפי דומיין — גם לפי נתיב. לדוגמה: רעשי השלמה אוטומטית בחיפוש Google מוסתרים, בלי לאבד נראות לחסימות אמיתיות באותו דומיין.

Not just by domain — by URL path too. Example: Google search autocomplete noise is hidden, without losing visibility of real blocks on the same domain.

➕

רשימה משופרת ע"י הקהילה

Community-improved list

ראית domain שצריך להוסיף לרשימת הפרסומות? לחץ על כפתור "הצע" בכרטיס החסימה — נפתח טופס GitHub עם פרטים מלאים. הוספה לרשימה מגיעה לכל המשתמשים תוך 24 שעות, ללא עדכון התוסף.

Spotted a domain that should be on the ad/tracker list? Click the "Suggest" button on the block card — a pre-filled GitHub form opens. Additions reach every installed copy within 24 hours, no extension update required.

⚡

עדכון יומי אוטומטי

Daily auto-refresh

הרשימה הציבורית של דומיינים ופרסומות מתעדכנת אחת ליום ברקע. תיקונים ושיפורים מגיעים אליך אוטומטית.

The public domain & tracker list refreshes once a day in the background. Fixes and additions reach you automatically.

📋

דוח להעתקה ושיתוף

Shareable report

לחץ על אייקון הדוח כדי להעתיק סיכום קריא של כל החסימות בדף — מקובץ לפי סוג, עם תאריך וכותרת. מצוין לשליחה בוואטסאפ, מייל או הודעה.

Click the report icon to copy a human-readable summary of every block on the page — grouped by type, with date and header. Great for WhatsApp, email, or a quick message.

⚙️

רשימה אישית בהגדרות

Personal allowlist

דף ההגדרות מאפשר להוסיף דומיינים משלך לרשימת ה"מותרים" — חסימות שלהם יוסתרו רק אצלך, מבלי להשפיע על משתמשים אחרים.

The options page lets you add your own domains to the "ignore" list — their blocks are hidden only for you, without affecting other users.

🔒

פרטיות קודם

Privacy first

הכל נעשה מקומית בדפדפן שלך. שום URL לא נשלח לשרתים שלנו, אין אנליטיקס, אין מעקב.

Everything runs locally in your browser. Not a single URL leaves your machine, no analytics, no tracking.

🌐

עברית ואנגלית

Hebrew & English

ממשק דו-לשוני מלא עם תמיכה ב-RTL. השפה נשמרת בין הפעלות וזוהית אוטומטית מהדפדפן.

Fully bilingual interface with RTL support. Language is remembered between sessions and auto-detected from the browser.

סוגי חסימות שהתוסף מזהה

Block types detected

🚫 רשימה שחורה 🚫 Blacklisted — האתר נסקר ונחסם במפורש — site reviewed and explicitly blocked
⏳ לא ברשימה הלבנה ⏳ Not whitelisted — ממתין לאישור / בסקירה — pending review
⚙️ הגדרות אישיות ⚙️ Personal settings — המשתמש חסם את זה בהגדרות שלו — user's own filter settings
📁 סוג קובץ 📁 File type — סוג קובץ שלא נתמך בסינון אוטומטי (zip/exe וכו׳) — file type not supported by automatic filtering (zip/exe/etc.)
❓ צד שלישי ❓ Third-party — תת-משאב שנחסם, סוג לא מזוהה — sub-resource block, type undetectable

רשימת הדומיינים המוסתרים

Hidden domains list

אלה הם דומיינים של פרסומות, אנליטיקה ומעקב שהתוסף מסתיר כברירת מחדל. חסימה שלהם לא משפיעה על תפקוד האתר. הרשימה מתעדכנת אחת ליום בכל התוספים המותקנים.

These are ad, analytics and tracking domains that the extension hides by default. Blocking them doesn't break the page. The list is auto-refreshed once a day on every installed copy.

עזור לשפר את הרשימה

Help improve the list

הדרך הקלה ביותר: בפופאפ של התוסף, בכל כרטיס חסימה יש כפתור הצע (⊕). לחיצה פותחת טופס GitHub עם הדומיין וכתובת הדף ממולאים מראש — נשאר רק להסביר למה הדומיין הוא פרסומת/מעקב ולשלוח.

Easiest way: every block card in the extension popup has a Suggest button (⊕). Clicking it opens a GitHub form pre-filled with the domain and the page URL — just explain why it's an ad/tracker and submit.
