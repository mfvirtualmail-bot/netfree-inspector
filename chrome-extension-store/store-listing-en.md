## Detailed Description

**NetFree Inspector** — The tool that tells you why the page isn't working.

### The Problem

NetFree is a whitelist-based content filter. When you open a website, sometimes parts of it don't load: images are missing, buttons don't respond, or sections of the page simply appear empty. Why? Because third-party resources the site depends on (scripts, images, API requests) are blocked by NetFree — but you can't see that directly, only the broken result.

Until now, to find out what was blocked, you had to open a support ticket, start a traffic recording, reload the page, and submit the recording for review. A long, tedious process.

### The Solution

NetFree Inspector passively monitors every request your browser makes and instantly detects any request blocked by NetFree (HTTP status 418). One click on the toolbar icon — and you see every block on the current page, organized in a clean, modern interface.

### What It Detects

🚫 **Blacklisted** — sites that were reviewed and explicitly blocked by NetFree

⏳ **Not Whitelisted** — unknown sites, pending review

⚙️ **Personal Settings** — sites blocked by your own NetFree personal settings

❓ **Third-Party Blocks** — sub-resources blocked on otherwise working pages

### Key Features

✓ **Automatic detection** of all blocks on the current page — no extra setup, no complex configuration

✓ **Direct support requests** — the "Open Request" button auto-generates a pre-filled NetFree whitelist ticket with the blocked URL and your current page as the referrer

✓ **Copy all blocked URLs** with one click — ready to paste anywhere

✓ **Red badge** on the toolbar icon showing the number of blocks on the current tab

✓ **Grouped by domain** — keeping the list clean and readable

✓ **Bilingual interface** — Hebrew (default) and English with full RTL support

✓ **Reload & Record** — refresh the page and capture all requests from scratch for complete documentation

### Complete Privacy

NetFree Inspector **does not collect any data**. All analysis happens locally in your browser. No information is sent to any external server. Block data is kept only in browser memory and cleared the moment you close the browser. No tracking, no analytics, no user accounts.

### Who It's For

• **Regular NetFree users** who want to understand why pages aren't working correctly
• **Technical support staff** who need to diagnose issues in internal systems
• **Web developers** working with users behind the filter who need to know which resources are blocked

### How to Use

1. Install the extension
2. Browse normally
3. When a page isn't working correctly, click the NetFree Inspector icon in the toolbar
4. Instantly see all blocks on the page
5. Click "Open Request" to submit directly to NetFree, or "Copy" to get the URLs
